Sport Spots

The SportSpots mobile application aims to make it easier for outdoor sports enthusiasts to find “spots” suitable for their favorite sports via a google map. The application is community-based because users who have created an account can add, verify, rate and comment on spots.

This application is made with React Native, FireBase and the Google Map API.



CineMates

CineMates is a website that use the web scraping to give a mark to every movies. The website browse the internet and recovers on multiple other website the marks given by the users. The algorithm make the average depending on the number of votes and display the perfect average mark.

CineMates includes also an other useful tool for the cinephile like an randomized film proposition based on your preferences.

CineMates is a Python Web site made with Django. I used the Python libarie BeautifulSoup4 and Ajax for the requests between frondend and backend.



Alphonse Bot

Alphonse is a small twitter bot that post tweets everyday. For now, he create "Life is ..." sentences with a random word at the end. Due to the randomness of his sentence, he often talks nonsense but we love him anyway.

He is made in Python with the Twitter API and the module Tweepy. He is currently hosted on a Raspberry Pi 4.

Go to his profile : https://twitter.com/Alphonse_bot



SkyAirLine Corp

SkyAirLine Corporation is a real physics-based airplane game where you can roam freely, extinguish forest fires in Fire Fighting Airplane mode, deliver packages or even do stunts with an aerobatic plane.

This game was made in Unity3D with a team of 7 developpers or 3D modelers. We used the Agile method and I was the Scrum Master of this project.

AlphonseBot

maintenance explanation :

During this maintenance, I add a new feature. Alphonse can know reply to tweets that mention him. When you summon him on a tweet you can put a hashtag in your tweet. Alphonse will recovers your tweet, extract the hashtag and analyse for you this trend. For that, he use a really powerfull algorithm of sentiment analysis. He will get the last 250 tweets on this specific hashtag and deduce the current atmosphere.

Technological watch :

During this maintenance, I didn't have many choices. I had to use the python module Textblob. This module can do many things but I only use it for the "sentiment polarity" function that return a float between -1 and 1. The closer the value is to 1, the more cordial the text and the closer the value is to -1, the more hateful the text.
I use this value to create differents sentences for Alphonse.
